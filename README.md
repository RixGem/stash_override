# Stash Overrides

A collection of Stash override configurations for various purposes.

## Overrides

| File | Name | Description | Category |
|------|------|-------------|----------|
| `AI.override` | Apple Intelligence Proxy | Forces Apple Intelligence, Siri, and Private Cloud Compute domains through the Proxy group to bypass geo-restrictions. | AI |
| `apple-relay-proxy.override` | Apple Relay Proxy | Override for Apple Relay services to use proxy. | Privacy |
| `prevent-dns-leak.override` | Prevent DNS Leak | Adds a rule to prevent DNS leaks and enables fake-ip mode. | Security |
| `wechat.override` | WeChat Proxy | Forces WeChat International domains to route through the Proxy group. | Social |

## Usage

Import these files into your Stash configuration overrides.
